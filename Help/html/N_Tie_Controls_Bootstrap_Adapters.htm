Tie.Controls.Bootstrap.Adapters Namespace
The Tie.Controls.Bootstrap.Adapters namespace contains WebControlAdapters for applying basic Bootstrap styling to select ASP.NET user controls.

InputAdapter
Adapter to apply Bootstrap classes to a base ASP.NET input (e.g. TextBox, DropDownList, or ListBox).
